# Multi-Agent-System-Crewai

### 1. Financial Analysis Collaboration
  A multi-agent setup where financial domain agents (Data Analyst, Market Researcher, Report Writer, etc.) work together to:
  - Monitor and analyze market data  
  - Identify financial trends  
  - Generate structured reports to support decision-making  

---

### 2. Research Writer Assistant
  A team of agents (Planner, Content Writer, Reviewer) collaborating to:
  - Gather information on a given topic  
  - Draft structured research content  
  - Refine and produce a polished article
